    API

    Create

    To create a notice, make a post request

    /api/v1/user/{user_id}/notice

    The post body will be json data

    {"title": "titel", "text": "text"}

    Read

    To fetch a single notice

    /api/v1/user/{user_id}/{notice_id}/notice

    To fetch all notices by a user

    /api/v1/user/{user_id}/notices

    Delete

    To delete a notice, make a delete request

    /api/v1/user/{user_id}/{notice_id}/notice
